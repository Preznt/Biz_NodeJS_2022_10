### FormData

https://2ham-s.tistory.com/307

- ajax로 폼 전송을 가능하게 해주는 FormData 객체입니다.
- form을 제출하면 action 속성에 의해 지정한 페이지로 이동하면서 데이터를 전송합니다.
  ajax는 반대로 제출 버튼을 누르면 기본 폼 동작은 e.preventDefault() 로 멈추고, 페이지 전환 없이 데이터를 전송
